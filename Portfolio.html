RCSB - Research Collaboratory for Structural Bioinformatics

RCSB was my first foray into Front-End Team Lead. I worked with a small team of Junior Developers to implement new features, maintain legacy code, bug fix, work with 3rd party services, and otherwise keep RCSB.org and PDB101.RCSB.org in tip-top shape.

RCSB was largely concerned with funcationality and providing the scientific community (our primary user base) the best features with the most complete data. UI/UX was on the drawing board, but did not have priority over improving the data and functionality.

Atlassian Community

Community.Atlassian.com was a fairly robust legacy project. Built on top of the Khoros platform, Khoros provides a lot of the fundamental forum features that we know and expect from these sorts of websites. Atlassian has a lot of ideas about what should be provided to users, so the implementation and maintenance of numerous features, such as gamification, notifications, 3rd party integrations were all part and parcel.

Takeda Clinical Trials

Working with everyone at Thorium Digital, we put together this full stack project. A full stack project, I primarily worked on UI and UX implementations. Filling in gaps in the design documents and providing the user with a seamless experience on all device sizes. I also handled a few prototype features (mostly UX flourishes) that didn't end up making the cut, as they didn't sit well with users.

Baileys

While at Thorium I was the primary developer for content updates on Baileys.com. Which was a little sobering as a Junior Developer, but pretty fun to work on!

Kikori Whiskey

Kikori is a Japanese rice whiskey. Originally Wordpress, I designed a custom scraper to pull the content into JSON and process it for the Gulp/Nunjucks engine to consume. I handled nearly every step in the process.

Join The Pact

Join The Pact is a site for people to pledge against drunk driving. I was responsible for website architecture, structure, localization, and database design. I translated UI mockups into HTML and CSS using the Gulp/Nunjucks engine.

Quaker City Shrubs

QC Shrubs is a brand of malt beverage, with a distinct site meant to convey its distinct flavor. I was responsible for website architecture, structure, UI and UX development. I translated UI mockups into HTML and CSS using the Gulp/Nunjucks engine.

Cascade Blonde

Cascade Blonde is a brand of Tennessee whiskey. For their site I handled translating the mockups into HTML, CSS, and Javascript animations. This site was a lot of fun to work on, with a nice color pallette and interesting layout.
